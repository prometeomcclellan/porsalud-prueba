# porsalud-prueba
Nombre del proyecto : Prueba Por Salud
Nombre del app      : Fotos Por Salud
Versión             : 0.0.1
Autor               : Omar McClellan
Framework           : ionic/angular
Herramientas        : Visual Studio Code, Android Studio, Windows Command Prompt, Windows navegador de archivos y web
Descripción         : App que permite el uso de la cámara local para tomar una fotografía y almacenrla en el dispositivo a su vez agregando una breve descripción a manera de leyenda  de la imagen. En su defecto si no existe una cámara reconocible o disponible en el dispositivo en uso, se accede a la galería interna del mismo y se obtiene     y se muestra la magen seleccionada en un contenedor al que se puede agregar la descripción a manera de leyenda



Modo de instalación : para instalar el presente proyecto se debe seguir el siguiente orden
1. Preparación del entorno de trabajo. Para preparar tu entorno de trabajo en este proyecto deberás :
    a. Descargar e instalar el Java JDK8 desde https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html
    b. Descargar e instalar Android Studio desde https://developer.android.com/studio
    c. Descargar e instalar Node js desde https://nodejs.org/en
    d. Descargar e instalar Visual Studio Code desde https://code.visualstudio.com
    e. Descargar e instalar Git desde https://git-scm.com/download/win
    f. Crear una cuenta en Github (Si aún no la tienes) desde https://docs.github.com/es/desktop/installing-and-configuring-github-desktop/installing-and-authenticating-to-github-desktop/installing-github-desktop

    NOTA.- en todas y cada una de estas instalaciones asegurate de que el instalador tenga la opción de agregar el comando principal de cada framework a la variable global de entorno o PATH, asi podrás utilizar los comandos que a continuación se mencionarán

2. Preparación e instalación del framework ionic/angular. Para este fin deberás :
    a. Crear una carpeta en tu directorio local de preferencia, el nombre de la carpeta sugerido es 'Por Salud'
    b. Acceder a la carpeta recién creada (cd Por Salud)
    c. Abrir la command prompt (Windows)/terminal (iOs)
    d. Desde la línea de comando, inicializar Git con el comando git init
    e. Desde la línea de comando, clonar el proyecto con el comando git clone https://github.com/prometeomcclellan/porsalud-prueba.git
    f. Desde la línea de comando, acceder a la carpeta del proyecto con el comando cd C:\Users\Omar\Desktop\Desarrollo\Por Salud\porsalud-prueba\porsalud-cam
    g. Desde la línea de comando, instalar ionic, podrás seguir la guía de instalación desde https://ionicframework.com/docs/intro/cli (De preferencia hz la instalació global con -g)
    h. Desde la línea de comando, instalar el proyecto con el comando npm install. Esto instalara las siguientes dependencias
